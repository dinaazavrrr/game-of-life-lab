## Інструкції для перевіряючого

Цей проект реалізує алгоритм "Гра Життя" (Conway's Game of Life) на JavaScript.

### Налаштування середовища (Крок 1)

Для налаштування середовища та встановлення залежностей виконайте одну команду:

```bash
npm install
```
(Ця команда встановить залежності (Jest) з файлу package.json)

### Запуск тестів (Крок 2)
Для запуску всіх юніт-тестів, які покривають правила гри, граничні умови та приклади симуляції, виконайте:

```bash
npm test
```
(Успішний результат демонструє, що логіка гри працює коректно.)

### Запуск програми (Крок 3)
Програма зчитує вхідні дані з файлу input.txt та записує результат симуляції у файл output.txt.

Щоб запустити симуляцію, виконайте:

```bash
npm start
```
(Після виконання, перевірте вміст output.txt.)
